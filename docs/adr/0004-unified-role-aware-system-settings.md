# 使用统一且按角色展示的系统设置

账户、个人偏好和管理员能力统一放入图片工作区内的全屏系统设置界面，采用左侧
分组导航和右侧内容区；普通用户只看到个人设置，管理员在同一界面额外看到用户、
部门资源、配额、调度、内容查看和审计功能。账户入口固定在工作区侧栏底部，旧
`/admin` 地址只作为进入管理员设置的兼容入口，不再维护另一套管理产品界面。
这样既保留原图片工作区作为产品主界面，也避免账户与管理功能散落在顶部按钮、
设置弹窗和独立后台之间。

## Consequences

- 系统设置使用应用内全屏双栏布局，并按当前用户角色限制菜单、搜索结果与接口。
- 普通偏好即时保存；凭据、权限和管理员配置显式确认后生效。
- 离开未保存表单时必须拦截，高影响操作必须二次确认；Web 界面不提供物理删除。
- 原服务器路径配置不再作为用户设置展示，存储页改为显示个人用量与额度。
